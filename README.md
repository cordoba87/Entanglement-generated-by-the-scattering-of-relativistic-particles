# Entanglement-generated-by-the-scattering-of-relativistic-particles
Bachelor's degree in physics final project.
In order to use the codes on your own mathematica and replicate or expand the results: just download the nb. files and run the code, in order to change the initial helicity state search for the l1=h1 and l2=h2 line and change it. I would highly reccommend to run the code step by step so the reader can understand the development of the results obtained,as well as the limits and figures that are generated throughout the code.
Use "solo ms" for the soft supersymmetry breaking generated by scalar mass.
Use "mf y ms" for the soft supersymmetry breaking generated by fermionic mass.
All of the codes work on all mass limits.
